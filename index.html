A graph-based application that models an entire railway network as a weighted graph and uses Dijkstra's algorithm to compute the shortest and most efficient route between any two stations. It supports dynamic addition of stations and routes, and visualizes the network interactively so users can see the optimal path highlighted in real time.

  • Interactive graph visualization

Compiler Simulator

A mini compiler that replicates the front-end phases of a real compiler. It uses Lex for lexical analysis to break source code into tokens, and Yacc for syntax parsing to build a parse tree based on a custom grammar. The tool outputs token streams, parse trees, and basic semantic analysis results — demonstrating how programming languages are processed under the hood.

  • Lexical analysis with Lex
  • Syntax parsing with Yacc
  • Token stream & parse tree output
Lex Yacc C
